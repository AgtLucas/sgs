Starting

Import the sgs.styl file:


    @import "path/to/sgs"

Semantic Usage (#1 method)

In #1 method, the responsive design has to be made manually.


    .my-div
      column(8)
      @media tablet
       column(7)

    .other-div
      column(4)
      @media tablet
       column(5)

Result:

.my-div
.other-div

Semantic Usage (#2 method)

In #1 method, the responsive is automatically. Is used @extend stylus method.


    .my-div
      @extend .sgs8

    .other-div
      @extend .sgs4

Result:

.my-div
.other-div

Non-Semantic Usage

Is used classes from 1 to 12 (max colums configurable) for markup in your html. The sgs name also is configurable.


    <div class="container">
        <div class="sgs3">3</div>
        <div class="sgs3">3</div>
        <div class="sgs3">3</div>
        <div class="sgs3">3</div>
    </div>

Result:

3
